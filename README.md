# 🚀 我的第一个网页 - My First Website

这是一个精心设计的交互式网页项目，展示了 HTML、CSS 和 JavaScript 的基础应用。适合作为学习 Web 开发的起点。

## ✨ 功能特性

- **响应式设计**：完美适配桌面端和移动端设备
- **流畅动画效果**：包含入场动画、弹跳表情等视觉特效
- **交互式功能**：
  - 💬 消息提示系统
  - 🔢 点击计数器
  - 🎨 表情符号切换
  - ⌨️ 键盘交互（空格键计数）
- **现代化设计**：渐变背景、卡片式布局、阴影效果
- **用户体验优化**：悬停效果、平滑过渡、自动消失提示

## 🎨 界面预览

### 主要元素
- **渐变背景**：紫色系渐变（#667eea → #764ba2）
- **白色卡片**：圆角设计，带阴影效果
- **动态表情**：8种可选表情符号循环切换
- **功能列表**：展示5项核心特性
- **三个交互按钮**：
  - 蓝色按钮 - 显示消息
  - 绿色按钮 - 计数器
  - 红色按钮 - 切换表情

## 🛠️ 技术栈

- **HTML5**：语义化标签结构
- **CSS3**：
  - Flexbox 布局
  - CSS 动画（@keyframes）
  - 媒体查询（响应式设计）
  - 渐变背景
  - 过渡效果
- **JavaScript**：
  - DOM 操作
  - 事件监听
  - 定时器（setTimeout）
  - 键盘事件处理
